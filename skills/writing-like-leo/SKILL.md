# Writing like Leo

Use this skill before writing or substantially editing text in Leo's voice.

## Workflow

1. Read `references/WRITING.md`.
2. Choose the relevant mode.
3. For rewriting or humanizing existing text, read `references/writing/rewriting.md`.
4. For original writing from notes, sources, or experiments, read `references/writing/original.md`.
5. If a sample of Leo's writing is available, match the sample before applying generic guidance.
6. Preserve factual claims and uncertainty from the source material.
7. Before finalizing, scan for the non-negotiables in `WRITING.md`.

## Constraints

Keep the output appropriate to the medium.
Do not force personality into reference, legal, or technical text when plain neutral prose is better.
For long Markdown files, keep one full sentence per physical line.

# Third-party skills

This repository vendors selected third-party skills so Leo's dotfiles can install one stable skills source.

## Vendored sources

- `skills/domain-modeling` comes from `mattpocock/skills`, path `skills/engineering/domain-modeling`.
- `skills/grilling` comes from `mattpocock/skills`, path `skills/productivity/grilling`.
- `skills/skill-creator` comes from `anthropics/skills`, path `skills/skill-creator`.

## Licenses

- Matt Pocock skills license: `licenses/mattpocock-skills-LICENSE`.
- Anthropic skill creator license: `licenses/anthropics-skill-creator-LICENSE.txt`.

When updating vendored skills, replace the vendored directory from upstream and refresh the matching license file if upstream changed it.
